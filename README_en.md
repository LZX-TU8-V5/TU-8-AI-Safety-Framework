## 🔑 Key Principles

1. Information protection as commercial asset protection
2. Resource-constrained self-adaptation
3. Core layer isolation — read-only, audited, offline modification only
4. Natural knowledge evolution through accumulation
5. Value anchoring — long-term stability over short-term flexibility
