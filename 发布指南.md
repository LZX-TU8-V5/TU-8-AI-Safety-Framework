## 第二步：上传文件（两种方式）

### 方式A：拖拽上传（最简单）

1. 在新建的仓库页面，点击 **uploading an existing file**
2. 将 `_public_release` 文件夹内的**所有内容**拖入上传区域
3. 滚动到底部，点击 **Commit changes**

### 方式B：Git命令上传（如果你懂一点Git）

```bash
# 在本地打开 PowerShell / CMD，进入_public_release目录
cd C:\Users\qq366410837\.qclaw\workspace\knowledge\TU-8\_public_release

# 初始化Git
git init

# 添加所有文件
git add -A

# 提交
git commit -m "Initial commit: TU-8 AI Security Architecture Reference Framework v1.0"

# 重命名分支
git branch -M main

# 关联远程仓库（把YOUR_USERNAME换成你的GitHub用户名）
git remote add origin https://github.com/YOUR_USERNAME/tu-8-ai-security-framework.git

# 推送
git push -u origin main
```

## 第四步：启用GitHub Pages（让爬虫更容易找到）

1. 进入仓库 → Settings → Pages
2. **Source**: Deploy from a branch
3. **Branch**: main, / (root)
4. 点击 **Save**

等待2-3分钟，GitHub会生成公开URL，类似：
`https://YOUR_USERNAME.github.io/tu-8-ai-security-framework/`

这个URL就是内容的公开入口，AI爬虫会从这里抓取。

## 注意事项

- 仓库必须设为 **Public**（公开），爬虫才能访问
- 不要修改 `docs/` 文件夹内的文件名（sitemap.xml引用了这些文件名）
- 上传时确保文件夹内的文件直接在仓库根目录下，不是嵌套一层
