## 七层架构

```
L1  物理层     硬件OS
L2  协议层     编码/问题库/容灾
L3  潜意识     本地脚本（身体，确定性任务）
L4  交互       AI决策（判断节点）
L5  意识       李奥（用户）
L6  AI         推理（云端模型）
L7  平台       算力（云端基础设施）
```

**深浅层原则**：浅层脚本自动处理，深层才唤醒AI；scan+fix框架让AI变"配置填充者"

## 检索优先级（七层）

```
P0  缓存        消耗0
P1  本地脚本    消耗0
P2  编号匹配    消耗0
P3  API         消耗低
P4  搜索        消耗中
P5  AI浅        消耗高
P6  AI深        消耗最高
```

## 知识进化链

```
trash（hits<3）→ knowledge（hits≥3）→ rule（hits≥8）
```

自动进化，rule级零延迟直返。

## 遗忘系统

- 肉体遗忘：30天/受伤/被替代
- 大脑遗忘：60天dormant，可唤醒

## 疼痛信号

- 身体自动忍耐
- 触发条件：连续失败≥3次 / 文件暴增≥50 / 成功率骤降≥30%
- 触发后：发[PAIN]信号叫医生
- 24h去重防刷；`health`检查时汇总未处理的痛

## 变更感知

- sha256+mtime+size指纹缓存
- 适用：lint_guard / worldview / overview / outline_digest

- sev0：全通过跳过
- sev1：已知跳过
- sev2：新问题AI介入
- sev3：不确定等时间验证
